Python3 学习

发表于 2018-03-04 | 分类于 Linux

Python 优点
开发效率高，强大的库
Python 缺点
运行速度慢，代码不能加密

用 exit() 或 ctrl+d 退出 python 交互模式

在 xx.py 首行加上 #!/usr/bin/env python3 并加上执行权限可直接运行 xx.py 文件
python 允许用 r’’ 表示 ‘’ 内部的字符串默认不转义，允许用 ‘’’…’’’ 的格式表示多行内容

print 函数会触发一个换行操作

int 函数丢弃小数部分转换成整数，round(n,r) 函数四舍五入保留小数点后 r 位

当数字 n 是两个相邻整数的中间值（例如 1.5、2.5) 时，round(n) 返回其最为接近的偶数

Python 中可以使用整除运算符 //

Python 通过垃圾回收进程将孤立的数值从内存中移除

在 IDLE 编辑器中，按下 Alt+3 和 Alt+4 键可以注释或不注释选注的代码段

Python 不允许字符串中单个字符的索引越界，但是在切片中可以允许索引越界

print(dir(str)) 会显示所有的字符串方法

print 有两个可选参数 seq 和 end

列表是可变对象，数值、字符串和元组是不可变对象

list 函数可以将元组或字符串转换为列表

tuple 函数可以将列表或字符串转化为元组

sort 没有返回值，list = list.sort() 是无效语句

在函数定义中，没有默认值的参数必须放在有默认值的参数前面

按位置传递的实际参数必须放在按关键字传递的实际参数前面
