Python3 学习

Python 优点
开发效率高，强大的库
Python 缺点
运行速度慢，代码不能加密

用 exit() 或 ctrl+d 退出 python 交互模式

在 xx.py 首行加上 #!/usr/bin/env python3 并加上执行权限可直接运行 xx.py 文件

python 允许用 r’’ 表示 ‘’ 内部的字符串默认不转义，允许用 ‘’’…’’’ 的格式表示多行内容

CentOS httpd中文文件名乱码

  1. 检查语系
    echo $LANG

  2. vim /etc/conf/httpd/httpd.conf
    添加 IndexOptions Charset=UTF-8

  3. 重启httpd
    service httpd restart
